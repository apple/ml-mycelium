Namespace ui

Provides builders to construct contents of Node and Tooltip.

  nodeId,
  new ui.VStack(
    new ui.Text('Node').with({ fontWeight: 600 }),
    new ui.Text(nodeId),
  ),
).with({
  backgroundColor: ui.Theme.GRAY_BACKGROUND,
  tooltip: () => new ui.Text('This is a new'),
});
